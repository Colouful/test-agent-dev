# 09 - 样式规范

## 技术选型

- 主要样式方案：**Tailwind CSS v4**（或 UnoCSS，视项目初始化决定）
- 组件库：**Element Plus**（Vue 3 官方生态）或裸 Tailwind 组件
- 全局样式：`src/assets/main.css`（只放 CSS 变量和全局重置）

## Tailwind 使用规范

- 优先使用 Tailwind 工具类，不新增自定义 CSS
- 复杂的条件类合并使用 `clsx` 或 Vue 的 `:class` 对象语法：

```vue
<!-- Vue :class 对象语法（推荐）-->
<div :class="['card', { 'card--active': isActive, 'card--disabled': disabled }]">

<!-- 或用 clsx（复杂场景）-->
<div :class="clsx('card', isActive && 'card--active')">
```

- 响应式前缀顺序：`sm:` → `md:` → `lg:` → `xl:`
- 状态前缀：`hover:` `focus:` `disabled:` `dark:`

## CSS 变量（主题 token）

```css
/* assets/main.css */
:root {
  --color-primary: 221.2 83.2% 53.3%;
  --color-surface: 0 0% 100%;
  --color-text: 222.2 84% 4.9%;
  --radius-card: 0.5rem;
}

.dark {
  --color-surface: 222.2 84% 4.9%;
  --color-text: 210 40% 98%;
}
```

- 禁止在组件里硬编码颜色值（如 `text-[#333]`、`style="color: red"`）
- 使用语义化 token（`text-primary`、`bg-surface`）

## 布局规范

- 页面最大宽度：`max-w-3xl`（详情/表单页）/ `max-w-6xl`（列表页）
- 页面内边距：`px-4 py-6`（移动）→ `sm:px-6 lg:px-8`（桌面）
- 间距使用 4 的倍数（Tailwind 默认单位：`gap-4`、`mt-6`）
- 卡片内边距：`p-4`（密集）/ `p-6`（宽松）

## 暗色模式

- 通过 `class` 策略（在 `<html>` 元素添加/移除 `.dark` 类）
- 切换逻辑放 `useThemeStore`（Pinia）
- 所有新增颜色必须同时定义亮/暗两套值

## 禁止事项

- 禁止内联 `style` 属性（动态宽高等特殊值除外）
- 禁止 `!important`
- 禁止直接覆盖 Element Plus 组件的 scoped CSS（用 `:deep()` 或 CSS 变量覆盖）
